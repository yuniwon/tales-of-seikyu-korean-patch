# Tales of Seikyu 비공식 한글패치 설치/업데이트/제보 안내

안녕하세요.

Tales of Seikyu 비공식 한글패치 배포 안내입니다.

## 1) 최신 버전

- 패치 버전: `0.1.10-playtest.20260702`
- 지원 플랫폼: Steam Windows
- 대상 게임 버전/buildid: Steam buildid `23980115`
- 최근 갱신일: 2026-07-02

## 2) 다운로드

GitHub Releases에서 `TalesOfSeikyuKoreanPatch-v0.1.10-playtest.20260702.zip`을 다운로드해 주세요.

다른 곳으로 공유할 때는 파일 재업로드 대신 이 저장소/릴리스 링크를 공유해 주세요.

## 3) 설치 방법

1. ZIP 파일을 다운로드합니다.
2. 원하는 위치에 압축을 풉니다.
3. `TalesOfSeikyuKoreanPatch.exe`를 실행합니다.
4. 첫 화면에서 게임 경로, 패치 상태, 폰트 상태, 최신 패처 상태를 확인합니다.
5. 게임 폴더가 자동으로 잡히지 않으면 `게임 폴더 찾기`로 설치 폴더를 지정합니다.
6. `최신 패처 확인`에서 새 버전이 보이면 `최신 패처 다운로드`를 눌러 새 ZIP을 받을 수 있습니다.
7. `한국어 패치 설치/업데이트`를 실행합니다.
8. 완료 후 게임을 다시 실행해 한글 적용 여부를 확인합니다.

최신 패처 다운로드는 ZIP 파일 저장까지만 진행합니다. 실행 중인 패처를 자동으로 덮어쓰거나, 다운로드한 EXE를 자동 실행하지 않습니다.

## 4) 제거(원본 복구) 방법

1. `TalesOfSeikyuKoreanPatch.exe`를 실행합니다.
2. `원본 복구`를 실행합니다.
3. 설치 시 생성된 백업으로 복구됩니다.

복원이 정상적으로 되지 않는다면 Steam 파일 무결성 검사를 사용해 주세요.

## 5) 이번 패처 수정 사항

- 기존 번역 22,488행은 유지했습니다.
- 가방 UI 오른쪽 정렬 카테고리 3개가 네모로 보이던 문제를 수정했습니다.
- 해당 3개 라벨은 글자 깨짐 방지를 위해 `Normal`, `Quest`, `Gold`로 표시합니다.
- 이전 패치가 이미 적용된 상태에서도 새 핫픽스 target과 다르면 다시 적용되도록 설치 로직을 보강했습니다.
- 게임이 업데이트되어도 그대로 맞는 문장은 자동으로 한국어를 적용합니다.
- 새로 추가되었거나 원문이 바뀐 문장은 건드리지 않고 원문으로 남깁니다.
- 바뀐 문장 목록은 패처가 제보 파일로 저장합니다.
- 부분 적용 상태에서도 `원본 복구`를 사용할 수 있게 했습니다.
- 글자 깨짐 방지 패치도 구조가 맞으면 자동으로 다시 적용합니다.
- 이전 설치 상태에서도 `한국어 패치 설치/업데이트`를 다시 누르면 가방 카테고리 표시 문제가 보강됩니다.
- 사용자 화면 문구를 더 쉽게 바꿨습니다. 예: `글자 표시 정상`, `대부분 적용됨`, `일부 문장은 원문일 수 있어요`.

## 6) 기존 패처 개선 사항

- 스타샌드 아일랜드 로컬 한글패치 배포본의 UX를 참고해, 상태 확인과 진단 기능을 보강했습니다.
- 첫 화면에 게임 경로, 패치 적용 상태, GitHub 최신 릴리스 상태를 표시합니다.
- 글자 표시 상태를 별도 카드로 표시합니다.
- Steam `libraryfolders.vdf` / `appmanifest_2340520.acf` 기반 설치 경로 자동 탐색을 보강했습니다.
- 패처에서 Steam을 통한 게임 실행 요청을 할 수 있습니다.
- 문제 제보용 진단 리포트 저장 기능을 추가했습니다.
- 게임 폴더를 직접 수정하지 않고 패치된 파일을 별도 폴더에 생성하는 오프라인 출력 기능을 추가했습니다.
- GitHub 최신 릴리스 확인과 ZIP 다운로드 기능을 추가했습니다.
- 다운로드 완료 후 저장 폴더를 열어 바로 새 ZIP을 찾을 수 있게 했습니다.
- 작업 로그 저장 기능을 추가했습니다.
- 명령줄에서도 `--check-update`, `--download-update`, `--diagnose`, `--export-patch`, `--launch-game`을 사용할 수 있습니다.
